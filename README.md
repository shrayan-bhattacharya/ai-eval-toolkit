# AI Evaluation Toolkit

An automated tool for evaluating LLM outputs — scoring accuracy, 
completeness, and detecting hallucinations against source documents.

## What it does
- Scores LLM responses against ground truth data
- Detects hallucinations (fabricated facts, wrong numbers, unsupported claims)
- Generates evaluation reports (CSV + dashboard)
